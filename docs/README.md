# FES PML Macro Documentation

## Getting Started

### Requirements

- **AVEVA E3D Design 3.1.9 or later** is recommended for using the macros.
- The macros are currently not supported in **AVEVA Unified Engineering** (E3D 4.x.x).

### Installation

To set up the macros, follow these steps:

1. Download the latest version from the [Releases](https://auth.fes-hd.jp/releases) page (login required).
2. Extract the contents of the ZIP file.
3. Set the `PMLLIB` environment variable to the path of the `pmllib` directory within the extracted folder.
4. Set the `CAF_UIC_PATH` environment variable to the path of the `cafuic` directory within the extracted folder.

### Authentication

To use the macros, you must configure an access token on your PC. For more details, see the [PMLAuth](https://auth.fes-hd.jp) license authentication server page (login required).
